Student Feedback Project – Git Workflow

1️⃣ Create GitHub Repository

Go to GitHub → New Repo → Name it student-feedback → Copy the URL.

2️⃣ Initialize Git Locally

git init git add . git commit -m "Initial commit - Student Feedback Project"

3️⃣ Connect to GitHub

git remote add origin https://github.com/YOUR-USERNAME/student-feedback.git git branch -M main git push -u origin main

4️⃣ Push Future Updates

git add . git commit -m "Updated project files" git push

5️⃣ Clone Repository Elsewhere

git clone https://github.com/YOUR-USERNAME/student-feedback.git

6️⃣ Verify Synchronization

git pull

If nothing updates → Repositories are synced.

7️⃣ Pull Latest Before Editing

git pull origin main

8️⃣ Check Git Status

git status

9️⃣ View History

git log
